# Lib-Hooks Directory 
> This directory contains React hooks to interface with the Supabase API functions. 

### Responsibilities  
* Communication with Supabase for inventory and rental logic, user management. 
* Organization of frontend component and page code. 

### Design Notes
* Lib-hooks are custom-developed React hooks built upon common hooks like useState and useEffect to interface with the database functions. 

### Technologies 
* Standard React library (useEffect, useState)
* [Internal Supabase Lib](../database-functions/)
